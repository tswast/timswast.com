How many 3x3 images with 3 colors are there?

 Tuesday, November 21, 2017

There are 3⁹, or 19,683, 3x3 images with 3 colors. There are 9 pixels, each with 3 possible values. Since pixels arranged in a different order are a different image, we multiply 3 for each pixel.

What if the middle pixel is missing or always the same color?

I'm working on a project where I don't care about the center pixel because it will always be the same color. How many of these donut-shaped images are there?

There is only 1 possibility for the center pixel. There are still 3 possible colors for the remaining 8 pixels, so there are 3⁸, or 6,561, 3x3 donut-shaped images with 3 colors.

What if we consider a rotated image to be the same image?

If rotated images are considered the same, we know there are up to 4 possible rotations. We might be tempted to divide 6,561 by 4, but there are some images that are rotationally symmetric. This means dividing by 4 will give us too few images.
